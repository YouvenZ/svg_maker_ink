description: Optimised instructions for technical and informational diagrams.
category: preset
---
Create a clear, informative diagram with a logical layout and consistent visual language.

Guidelines:
- Use rectangles for processes, diamonds for decisions, and rounded rectangles for start/end nodes.
- Connect nodes with arrows (lines with arrowhead markers).
- Label every node and every connection arrow.
- Keep text legible — minimum effective font size 12 px.
- Align nodes on a grid for a clean, professional appearance.
- Use colour sparingly to group related elements or highlight key nodes.
- Add a brief title at the top of the diagram.
