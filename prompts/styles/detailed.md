---
name: Detailed
description: Rich, layered design with texture and visual complexity.
category: style
---
Create a richly detailed, visually complex design.

- Layer multiple shapes to create depth and interest.
- Include textures using hatching, noise, or pattern fills via SVG `<pattern>`.
- Use gradients to model light and shadow on surfaces.
- Add fine details such as highlights, reflections, or subtle line work.
- The overall result should reward close inspection.
